# 🧠 XGBoost Crypto Trading Bot

A fully automated AI crypto trading bot using:
- 🟢 XGBoost for market predictions
- 📈 RSI, SMA, MACD for signals
- 🛡️ Stop-loss & take-profit system
- 📬 Telegram alerts
- 🚀 Replit / server-ready

## ⚙️ Requirements
- Python 3.10+
- requests, xgboost, numpy, flask, pandas
